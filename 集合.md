#### :hatched_chick:将集合的接口与现实分离 ####

与现代的数据结构一样，Java集合类库也将接口与现实分离，首先看下我们所熟悉的数据结构--队列是如何分离的。有关队列我们就不多阐述，直接看下队列的接口：

```java
    public  interface  Queue<E>
    {
        void add(E element);
        E remove();
        int size();
    }
```

其中add方法为向队列添加元素，remove是删除一个元素。size是队列大小。
